MINIMAX TIC-TAC-TOE
Minimax Tic-Tac-Toe is a Flask-based implementation of the classic Tic-Tac-Toe game, utilizing the Minimax algorithm
to make the opponent AI more challenging, and using Alpha-Beta Pruning to make the algorithm more efficient.

I used Flask Python for the back-end development, and used HTML and CSS for the front-end development. I connected
this application to Heroku and connected the webpage to the diegojmejia.com domain.
